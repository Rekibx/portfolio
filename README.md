## 🚀 About the Project

This portfolio is a beautifully designed, fully responsive web application built with modern HTML, CSS, and JavaScript. It features a premium dark-mode aesthetic, micro-animations, glassmorphism design trends, and seamless responsiveness across devices.

### Features:
- **Modern UI/UX**: Crafted with a premium dark theme, neon glowing accents, and fluid typography.
- **Glassmorphism Elements**: Cards and UI components utilizing blur effects for a futuristic look.
- **Responsive Layout**: Designed utilizing CSS Grid and Flexbox to look stunning on mobile, tablet, and desktop screens.
- **Performance Optimized**: Lightweight code without any bulky frameworks.
- **Interactive Links**: Connect directly to my GitHub, LinkedIn, and Email.

## 🛠️ Technology Stack

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom properties, animations, transitions, grid/flexbox layouts.
- **JavaScript**: Minimalist scripts for dynamic link injection and interactivity.
